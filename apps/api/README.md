## Dev-среда

### Старт dev-сервера (http://localhost:3000):

`npm run strat:dev`

## Маршруты

### Страница SPA:

`GET /`

### Тестовый ответ API:

`GET /api/v1/test-api/`

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
